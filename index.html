¿Porqué de esta página web?

Se trata de una de las diferentes entregas que se deben realizar para completar el Bootcamp impartido por GeekHubs Academy.

En este caso se trata de la primera entrega, en la cual se debe realizar una página web que a su vez enlace con otras dos más. Como se puede observar está primera página "index.html" tiene una estructura sencilla pero compatible con diferentes tamaños de pantallas, es por ellos que se trata de una "web responsive"".

Los lenguajes utilizados para la construcción de esta web son HTML5 y CSS3. Existía la posibilidad de utilizar algún framework de estilos como:

Boostrap

Tailwind CSS

Bulma
